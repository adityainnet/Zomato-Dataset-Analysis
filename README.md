# Zomato Dataset Analysis

## Project Overview

This project analyzes the Zomato restaurant dataset to extract insights regarding restaurant ratings, cuisines, pricing, location preferences, and factors affecting customer satisfaction.

## Objectives

* Perform data cleaning and preprocessing
* Analyze restaurant ratings
* Identify popular cuisines
* Discover restaurant hotspots
* Study the relationship between cost, ratings, and customer engagement
* Generate business recommendations

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* WordCloud
* Google Colab

## Key Findings

* Most restaurant ratings lie between 3.5 and 4.2.
* North Indian cuisine is the most popular cuisine.
* BTM is the largest restaurant hotspot.
* Moderate-priced restaurants dominate the market.
* Restaurants with table booking facilities achieve higher ratings.
